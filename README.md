# BasicTextInterface
A text-based interface. Built using gcc, that is installed via Cygwin on Windows. 

# How to use
Windows

1. Install cygwin
2. Using cygwin, install gcc
3. Run the program in the cygwin command line interface (CLI).
4. That's it.
